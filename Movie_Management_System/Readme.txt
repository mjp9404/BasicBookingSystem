Project Title: Movie Management System

What the program does: The program allows a user to create a new record, retrieve a movie in a specific year, generate a number of movies randomly and delete a record.
Date: 2022-04-22

How to run the program: The display menu willbe displayed the moment you start the program. The options are(Add new movie, Print movies released in year, Print random list of movies and Delete a movie). The user has to input a number to select an option. If add movie is selected, the user will be asked to input the duration, title and year of the movie. If number two option is selected, the user will be prompted to input the movie year. Option three asks the user to input how many movies they want to be shown. The last option allows the user to delete a movie by entering the id.
