Title: Flight Reservation Management System.
Date: March 28,2022

What the program does: 
It allows users to book a flight and find its information. Finding flights is a search method where a user enters some details then returns a list of matching results.

List of deficiency: We could not display the result on the text field.
How to run the program: The user has two main choice(Flights and Reservations). The user can look for the flights under the Flights tab by entering the information(Flight Code,Departing Airport Code,Arrival Airport Code,Weekday,Time,Seats,Cost Per Seat). Reservation allows users to book a reservation. The user will be given the following(Reservation code, Flight code, Airline, Name, Citizenship and Cost).
